# Bricks Query Loop Variation Swatches

A WordPress plugin that adds a custom Bricks Builder element to display WooCommerce variation swatches on product query loops.

## Description

This plugin provides an elegant way to display product variation attributes (like colors, sizes, images) as swatches within Bricks Builder query loops. Perfect for shop archive pages, product grids, and custom product listings.

## Features

- **Auto-detection**: Automatically detects and displays the appropriate swatch type (Image → Color → Label)
- **Multiple swatch types**: Supports color swatches, image swatches, and label swatches
- **Customizable**: Full control over styling through Bricks Builder controls
- **Per-attribute settings**: Configure each attribute group independently
- **Min/Max value limits**: Control how many swatches to display
- **Conditional rendering**: Only render attribute groups when they meet minimum value thresholds

## Requirements

- WordPress 5.8+
- Bricks Builder 1.5+
- WooCommerce 5.0+

## Installation

1. Upload the plugin folder to `/wp-content/plugins/`
2. Activate the plugin through the WordPress 'Plugins' menu
3. In Bricks Builder, find the "Variation Swatches (Card)" element under the "MBM Woo" category

## Usage

1. Add the element inside a Query Loop set to Products
2. Select which attributes to display
3. Configure styling options as needed
